# Beego
本网站Golang1.14版本，使用Beego框架开发搭建完成。<br>
功能包括不限于书籍的分类搜索、浏览、评论、收藏和上传等功能，用户和管理员的注册管理。<br>
网站使用了nginx负载均衡，适配了高流量场景下的堵塞处理。
